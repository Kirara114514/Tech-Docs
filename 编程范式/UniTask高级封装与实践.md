еңЁејҖе§ӢеүҚпјҢжҲ‘д»¬йҰ–е…ҲйңҖиҰҒеӣһзӯ”дёҖдёӘж ёеҝғй—®йўҳпјҡ**дёәд»Җд№ҲжҲ‘д»¬иҰҒеҜ№ `UniTask` иҝӣиЎҢдәҢж¬Ўе°ҒиЈ…пјҹ** зӯ”жЎҲжҳҜпјҡ`UniTask` жҸҗдҫӣдәҶеә•еұӮзҡ„ејӮжӯҘж“ҚдҪңеҺҹиҜӯпјҢе°ұеғҸжҗӯд№җй«ҳзҡ„еҹәзЎҖз –еқ—гҖӮз„¶иҖҢпјҢзӣҙжҺҘдҪҝз”ЁиҝҷдәӣеҹәзЎҖз –еқ—еңЁеӨ§еһӢжҲ–еӨҚжқӮзҡ„йЎ№зӣ®дёӯеҸҜиғҪдјҡйҒҮеҲ°дёҖдәӣжҢ‘жҲҳпјҡ

### 1. жҸҗй«ҳжҳ“з”ЁжҖ§е’ҢејҖеҸ‘ж•ҲзҺҮ

-   **еҮҸе°‘йҮҚеӨҚд»Јз Ғпјҡ** и®ёеӨҡејӮжӯҘж“ҚдҪңжЁЎејҸеңЁйЎ№зӣ®дёӯдјҡеҸҚеӨҚеҮәзҺ°пјҢдҫӢеҰӮпјҡзӯүеҫ…дёҖж®өж—¶й—ҙгҖҒеҠ иҪҪж–Үд»¶гҖҒзҪ‘з»ңиҜ·жұӮзӯүгҖӮеҰӮжһңжҜҸж¬ЎйғҪд»ҺеӨҙзј–еҶҷ `UniTask.Delay`гҖҒ`await UniTask.SwitchToThreadPool()` зӯүпјҢдјҡдә§з”ҹеӨ§йҮҸйҮҚеӨҚдё”зӣёдјјзҡ„д»Јз ҒгҖӮ
    
-   **з»ҹдёҖжҺҘеҸЈпјҡ** `UniTask` жң¬иә«жңүеӨҡз§Қзӯүеҫ…ж–№ејҸпјҲжҢүж—¶й—ҙгҖҒжҢүеё§гҖҒзӯүеҫ…жқЎд»¶пјүпјҢйҖҡиҝҮе°ҒиЈ…еҸҜд»ҘжҸҗдҫӣдёҖдёӘжӣҙз»ҹдёҖгҖҒжӣҙиҜӯд№үеҢ–зҡ„жҺҘеҸЈгҖӮ
    
-   **йҡҗи—Ҹеә•еұӮз»ҶиҠӮпјҡ** е°ҒиЈ…еҸҜд»Ҙе°ҶзәҝзЁӢеҲҮжҚўгҖҒеҸ–ж¶Ҳд»ӨзүҢз®ЎзҗҶгҖҒејӮеёёеӨ„зҗҶзӯүеә•еұӮеӨҚжқӮжҖ§йҡҗи—Ҹиө·жқҘпјҢи®©дёҡеҠЎйҖ»иҫ‘ејҖеҸ‘иҖ…иғҪжӣҙдё“жіЁдәҺд»–д»¬иҰҒе®һзҺ°зҡ„еҠҹиғҪпјҢиҖҢдёҚжҳҜејӮжӯҘзј–зЁӢзҡ„з»ҶиҠӮгҖӮ
    

### 2. еўһејәд»Јз ҒеҸҜиҜ»жҖ§е’ҢеҸҜз»ҙжҠӨжҖ§

-   **иҜӯд№үеҢ–е‘ҪеҗҚпјҡ** йҖҡиҝҮе°ҒиЈ…пјҢжҲ‘д»¬еҸҜд»Ҙз”Ёжӣҙе…·дёҡеҠЎеҗ«д№үзҡ„еҗҚз§°жқҘе‘ҪеҗҚејӮжӯҘж“ҚдҪңпјҢдҫӢеҰӮ `DownloadAssetAsync`гҖҒ`FadeOutUI`пјҢиҖҢдёҚжҳҜжіӣжіӣзҡ„ `DoSomethingAsync`гҖӮ
    
-   **йҷҚдҪҺеҝғжҷәиҙҹжӢ…пјҡ** еҪ“дҪ зңӢеҲ° `await MyTaskUtils.WaitFor(2.5f)` ж—¶пјҢз«ӢеҲ»е°ұзҹҘйҒ“е®ғжҳҜеңЁзӯүеҫ…пјҢиҖҢдёҚйңҖиҰҒеҺ»жҖқиҖғ `UniTask.Delay` еҶ…йғЁеҸӮж•°зҡ„еҗ«д№үгҖӮ
    
-   **йӣҶдёӯдҝ®ж”№пјҡ** еҰӮжһңжңӘжқҘ `UniTask` жҹҗдёӘ API зҡ„дҪҝз”Ёж–№ејҸеҸ‘з”ҹеҸҳеҢ–пјҢжҲ–иҖ…йңҖиҰҒз»ҹдёҖдҝ®ж”№жҹҗдёӘејӮжӯҘиЎҢдёәпјҲдҫӢеҰӮз»ҹдёҖзҡ„ж—Ҙеҝ—иҫ“еҮәпјүпјҢеҸӘйңҖиҰҒдҝ®ж”№е°ҒиЈ…еұӮпјҢиҖҢдёҚйңҖиҰҒдҝ®ж”№жүҖжңүдҪҝз”ЁеҲ°иҜҘ API зҡ„ең°ж–№гҖӮ
    

### 3. жҸҗеҚҮд»Јз ҒеҒҘеЈ®жҖ§е’Ңе®үе…ЁжҖ§

-   **з»ҹдёҖејӮеёёеӨ„зҗҶпјҡ** ејӮжӯҘж“ҚдҪңдёӯзҡ„ејӮеёёеӨ„зҗҶеҫҖеҫҖе®№жҳ“иў«еҝҪз•ҘгҖӮе°ҒиЈ…еұӮеҸҜд»ҘеҶ…зҪ® `try-catch` еқ—пјҢзЎ®дҝқжүҖжңүејӮжӯҘд»»еҠЎзҡ„ејӮеёёйғҪиғҪиў«жҚ•иҺ·е’ҢеӨ„зҗҶпјҢйҒҝе…ҚзЁӢеәҸеҙ©жәғгҖӮ
    
-   **иҮӘеҠЁеҸ–ж¶Ҳз®ЎзҗҶпјҡ** `CancellationToken` жҳҜ `UniTask` зҡ„ж ёеҝғпјҢдҪҶжүӢеҠЁз®ЎзҗҶ `CancellationTokenSource` зҡ„з”ҹе‘Ҫе‘Ёжңҹе®№жҳ“еҮәй”ҷгҖӮе°ҒиЈ…еҸҜд»ҘиҮӘеҠЁз»‘е®ҡд»»еҠЎдёҺ `MonoBehaviour` зҡ„з”ҹе‘Ҫе‘ЁжңҹпјҢйҳІжӯўеҶ…еӯҳжі„жјҸе’Ңеғөе°ёд»»еҠЎгҖӮ
    
-   **зәҝзЁӢе®үе…Ёпјҡ** е°ҒиЈ…еҸҜд»ҘзЎ®дҝқж•Ҹж„ҹж“ҚдҪңпјҲеҰӮж–Үд»¶ IOпјүеңЁжӯЈзЎ®зҡ„зәҝзЁӢпјҲеҗҺеҸ°зәҝзЁӢпјүдёҠжү§иЎҢпјҢе№¶еңЁеҝ…иҰҒж—¶е®үе…Ёең°еҲҮеӣһдё»зәҝзЁӢпјҢйҒҝе…ҚеӨҡзәҝзЁӢи®ҝй—® Unity API зҡ„й”ҷиҜҜгҖӮ

## еҰӮдҪ•еҜ№ UniTask иҝӣиЎҢдәҢж¬Ўе°ҒиЈ…пјҲд»Јз ҒзӨәдҫӢдёҺи®ҫи®Ўж„Ҹеӣҫпјү

жҲ‘д»¬е°ҶйҖҡиҝҮеӣӣдёӘеёёз”ЁеңәжҷҜжқҘеұ•зӨәеҰӮдҪ•иҝӣиЎҢдәҢж¬Ўе°ҒиЈ…пјҢж¶өзӣ–дәҶе»¶иҝҹзӯүеҫ…гҖҒж–Үд»¶ IOгҖҒе‘ЁжңҹжҖ§д»»еҠЎе’Ңз”ҹе‘Ҫе‘Ёжңҹз»‘е®ҡгҖӮ

### 1. з»ҹдёҖзҡ„е»¶иҝҹзӯүеҫ…ж–№жі•пјҡ`MyTaskUtils.WaitFor`

иҝҷдёӘж–№жі•ж—ЁеңЁжҸҗдҫӣдёҖдёӘз»ҹдёҖзҡ„вҖңзӯүеҫ…вҖқеҠҹиғҪпјҢжҷәиғҪең°ж №жҚ®дј е…ҘеҸӮж•°йҖүжӢ©жҢүж—¶й—ҙжҲ–жҢүеё§зӯүеҫ…гҖӮ

**д»Јз Ғпјҡ**



```
using Cysharp.Threading.Tasks;
using System.Threading;
using System;
using UnityEngine;

public static class MyTaskUtils
{
    /// <summary>
    /// ејӮжӯҘзӯүеҫ…жҢҮе®ҡзҡ„ж—¶й—ҙпјҲз§’пјүжҲ–её§ж•°гҖӮ
    /// й»ҳи®ӨеңЁдё»зәҝзЁӢжӣҙж–°гҖӮ
    /// </summary>
    /// <param name="delayTime">зӯүеҫ…ж—¶й—ҙпјҲз§’пјүгҖӮеҰӮжһңе°ҸдәҺзӯүдәҺ 0пјҢеҲҷзӯүеҫ…жҢҮе®ҡеё§ж•°гҖӮ</param>
    /// <param name="delayFrames">еҰӮжһң delayTime <= 0пјҢеҲҷзӯүеҫ…жӯӨеё§ж•°гҖӮ</param>
    /// <param name="ignoreTimeScale">жҳҜеҗҰеҝҪз•Ҙ Time.timeScale зҡ„еҪұе“ҚгҖӮ</param>
    /// <param name="cancellationToken">еҸҜйҖүзҡ„еҸ–ж¶Ҳд»ӨзүҢгҖӮ</param>
    /// <returns>UniTaskгҖӮ</returns>
    public static async UniTask WaitFor(
        float delayTime,
        int delayFrames = 1,
        bool ignoreTimeScale = false,
        CancellationToken cancellationToken = default)
    {
        if (delayTime > 0)
        {
            await UniTask.Delay(TimeSpan.FromSeconds(delayTime), ignoreTimeScale, PlayerLoopTiming.Update, cancellationToken);
        }
        else
        {
            await UniTask.DelayFrame(delayFrames, PlayerLoopTiming.Update, cancellationToken);
        }
    }
}

```

**и®ҫи®Ўж„Ҹеӣҫпјҡ**

-   **еҚ•дёҖиҒҢиҙЈдёҺжҷәиғҪеҲӨж–ӯпјҡ** е°Ҷ Unity дёӯеёёи§Ғзҡ„жҢүж—¶й—ҙжҲ–жҢүеё§зӯүеҫ…йҖ»иҫ‘е°ҒиЈ…еҲ°еҚ•дёҖзҡ„ `WaitFor` ж–№жі•дёӯпјҢйҖҡиҝҮеҲӨж–ӯ `delayTime` жқҘжҷәиғҪйҖүжӢ©и°ғз”Ё `UniTask.Delay` жҲ– `UniTask.DelayFrame`пјҢз®ҖеҢ–дәҶи°ғз”Ёж–№зҡ„еҝғжҷәиҙҹжӢ…гҖӮ
    
-   **й»ҳи®ӨеҸӮж•°пјҡ** жҸҗдҫӣдәҶеҗҲзҗҶзҡ„й»ҳи®ӨеҸӮж•° (`delayFrames = 1`, `ignoreTimeScale = false`, `cancellationToken = default`)пјҢдҪҝеҫ—жңҖеёёи§Ғзҡ„вҖңзӯүеҫ…дёҖе°Ҹж®өж—¶й—ҙвҖқжҲ–вҖңзӯүеҫ…дёҖеё§вҖқзҡ„и°ғз”Ёйқһеёёз®ҖжҙҒпјҢдҫӢеҰӮ `await WaitFor(2.5f);` жҲ– `await WaitFor(0);`гҖӮ
    
-   **жҳҺзЎ®зҡ„ PlayerLoopTimingпјҡ** жҳҺзЎ®жҢҮе®ҡеңЁ `PlayerLoopTiming.Update` йҳ¶ж®өиҝӣиЎҢзӯүеҫ…пјҢиҝҷжҳҜжңҖеёёз”Ёзҡ„жёёжҲҸйҖ»иҫ‘жӣҙж–°ж—¶жңәгҖӮ
    
-   **дҝқз•ҷеҸҜеҸ–ж¶ҲжҖ§пјҡ** дҫқз„¶дҝқз•ҷдәҶ `cancellationToken` еҸӮж•°пјҢзЎ®дҝқе°ҒиЈ…еҗҺзҡ„ж–№жі•д»Қз„¶ж”ҜжҢҒд»»еҠЎеҸ–ж¶ҲпјҢиҝҷжҳҜ `UniTask` зҡ„ж ёеҝғдјҳеҠҝд№ӢдёҖгҖӮ

### 2. е®үе…Ёзҡ„ж–Үд»¶ IO ж“ҚдҪңпјҡ`FileIOTaskUtils`

ж–Үд»¶ IO жҳҜе…ёеһӢзҡ„ IO еҜҶйӣҶеһӢж“ҚдҪңпјҢе°Ҷе…¶ж”ҫеңЁеҗҺеҸ°зәҝзЁӢжү§иЎҢеҸҜд»ҘйҒҝе…Қйҳ»еЎһдё»зәҝзЁӢгҖӮ

**д»Јз Ғпјҡ**



```
using Cysharp.Threading.Tasks;
using System.IO;
using System.Text;
using System.Threading;
using UnityEngine;
using System;

public static class FileIOTaskUtils
{
    /// <summary>
    /// ејӮжӯҘиҜ»еҸ–ж–Үд»¶жүҖжңүж–Үжң¬еҶ…е®№гҖӮеңЁеҗҺеҸ°зәҝзЁӢжү§иЎҢгҖӮ
    /// </summary>
    /// <param name="filePath">ж–Үд»¶и·Ҝеҫ„гҖӮ</param>
    /// <param name="cancellationToken">еҸҜйҖүзҡ„еҸ–ж¶Ҳд»ӨзүҢгҖӮ</param>
    /// <returns>ж–Үд»¶зҡ„ж–Үжң¬еҶ…е®№пјҢеҰӮжһңеӨұиҙҘеҲҷиҝ”еӣһ nullгҖӮ</returns>
    public static async UniTask<string> ReadAllTextAsyncSafe(string filePath, CancellationToken cancellationToken = default)
    {
        string content = null;
        try
        {
            await UniTask.SwitchToThreadPool(); // еҲҮжҚўеҲ°еҗҺеҸ°зәҝзЁӢ
            cancellationToken.ThrowIfCancellationRequested(); // жЈҖжҹҘеҸ–ж¶Ҳ

            if (!File.Exists(filePath))
            {
                Debug.LogWarning($"ж–Үд»¶дёҚеӯҳеңЁ: {filePath}");
                return null;
            }

            content = await File.ReadAllTextAsync(filePath, cancellationToken);
        }
        catch (OperationCanceledException)
        {
            Debug.LogWarning($"иҜ»еҸ–ж–Үд»¶ж“ҚдҪңиў«еҸ–ж¶Ҳ: {filePath}");
            return null;
        }
        catch (Exception ex)
        {
            Debug.LogError($"иҜ»еҸ–ж–Үд»¶еӨұиҙҘ: {filePath} - {ex.Message}");
            return null;
        }
        finally
        {
            await UniTask.SwitchToMainThread(); // зЎ®дҝқеӣһеҲ°дё»зәҝзЁӢ
        }
        return content;
    }

    /// <summary>
    /// ејӮжӯҘеҶҷе…Ҙж–Үжң¬еҶ…е®№еҲ°ж–Үд»¶гҖӮеңЁеҗҺеҸ°зәҝзЁӢжү§иЎҢгҖӮ
    /// </summary>
    /// <param name="filePath">ж–Үд»¶и·Ҝеҫ„гҖӮ</param>
    /// <param name="content">иҰҒеҶҷе…Ҙзҡ„ж–Үжң¬еҶ…е®№гҖӮ</param>
    /// <param name="cancellationToken">еҸҜйҖүзҡ„еҸ–ж¶Ҳд»ӨзүҢгҖӮ</param>
    /// <returns>еҶҷе…ҘжҲҗеҠҹеҲҷиҝ”еӣһ trueпјҢеҗҰеҲҷиҝ”еӣһ falseгҖӮ</returns>
    public static async UniTask<bool> WriteAllTextAsyncSafe(string filePath, string content, CancellationToken cancellationToken = default)
    {
        bool success = false;
        try
        {
            await UniTask.SwitchToThreadPool();
            cancellationToken.ThrowIfCancellationRequested();

            string directory = Path.GetDirectoryName(filePath);
            if (!string.IsNullOrEmpty(directory) && !Directory.Exists(directory))
            {
                Directory.CreateDirectory(directory);
            }

            await File.WriteAllTextAsync(filePath, content, cancellationToken);
            success = true;
        }
        catch (OperationCanceledException)
        {
            Debug.LogWarning($"еҶҷе…Ҙж–Үд»¶ж“ҚдҪңиў«еҸ–ж¶Ҳ: {filePath}");
            success = false;
        }
        catch (Exception ex)
            {
                Debug.LogError($"еҶҷе…Ҙж–Үд»¶еӨұиҙҘ: {filePath} - {ex.Message}");
                success = false;
            }
        finally
        {
            await UniTask.SwitchToMainThread();
        }
        return success;
    }
}

```

**и®ҫи®Ўж„Ҹеӣҫпјҡ**

-   **зәҝзЁӢе®үе…Ёпјҡ** йҖҡиҝҮеңЁ `try` еқ—ејҖе§Ӣж—¶ `await UniTask.SwitchToThreadPool()`пјҢе№¶еңЁ `finally` еқ—дёӯ `await UniTask.SwitchToMainThread()`пјҢзЎ®дҝқдәҶж•ҙдёӘ IO иҝҮзЁӢзҡ„зәҝзЁӢе®үе…ЁжҖ§пјҢе№¶иғҪеңЁж“ҚдҪңе®ҢжҲҗеҗҺе®үе…Ёең°еӣһеҲ°дё»зәҝзЁӢпјҢйҒҝе…Қйҳ»еЎһ Unity дё»зәҝзЁӢгҖӮ
    
-   **еҒҘеЈ®жҖ§дёҺй”ҷиҜҜеӨ„зҗҶпјҡ**
    
    -   еҶ…зҪ® `try-catch` еқ—пјҢиғҪеӨҹжҚ•иҺ· `OperationCanceledException` е’Ңе…¶д»– `Exception`пјҢе№¶еңЁжҺ§еҲ¶еҸ°иҫ“еҮәиӯҰе‘Ҡ/й”ҷиҜҜдҝЎжҒҜпјҢйҳІжӯўжңӘеӨ„зҗҶзҡ„ејӮеёёеҜјиҮҙзЁӢеәҸеҙ©жәғгҖӮ
        
    -   `ReadAllTextAsyncSafe` еўһеҠ дәҶ `File.Exists(filePath)` жЈҖжҹҘпјҢйҒҝе…Қе°қиҜ•иҜ»еҸ–дёҚеӯҳеңЁзҡ„ж–Үд»¶гҖӮ
        
    -   `WriteAllTextAsyncSafe` еңЁеҶҷе…ҘеүҚдјҡжЈҖжҹҘе№¶иҮӘеҠЁеҲӣе»әзӣ®ж Үж–Үд»¶зҡ„зҲ¶зӣ®еҪ•пјҢеўһеҠ дәҶдҫҝеҲ©жҖ§гҖӮ
        
-   **зӣҙи§Ӯзҡ„иҝ”еӣһеҖјпјҡ** `ReadAllTextAsyncSafe` иҝ”еӣһ `string` жҲ– `null` (иЎЁзӨәеӨұиҙҘ)пјҢ`WriteAllTextAsyncSafe` иҝ”еӣһ `bool` (иЎЁзӨәжҲҗеҠҹжҲ–еӨұиҙҘ)гҖӮиҝҷз§Қи®ҫи®Ўи®©и°ғз”Ёж–№жӣҙе®№жҳ“йҖҡиҝҮиҝ”еӣһеҖјеҲӨж–ӯж“ҚдҪңз»“жһңпјҢиҖҢж— йңҖејәеҲ¶дҪҝз”Ё `try-catch`гҖӮ
    
-   **е®Ңж•ҙеҸҜеҸ–ж¶ҲжҖ§пјҡ** жҜҸж¬Ўж“ҚдҪңеүҚйғҪ `cancellationToken.ThrowIfCancellationRequested()`пјҢе№¶е°Ҷе…¶дј йҖ’з»ҷеә•еұӮзҡ„ IO ж–№жі•пјҢзЎ®дҝқй•ҝж—¶й—ҙиҝҗиЎҢзҡ„ IO ж“ҚдҪңд№ҹиғҪиў«еҸҠж—¶еҸ–ж¶ҲгҖӮ
    

----------

### 3. еҸҜжҺ§зҡ„е®ҡж—¶еҷЁе’ҢеҫӘзҺҜпјҡ`LoopTaskUtils.DoPeriodicTask`

иҝҷдёӘе°ҒиЈ…жҸҗдҫӣдәҶзҒөжҙ»зҡ„е‘ЁжңҹжҖ§д»»еҠЎжү§иЎҢиғҪеҠӣпјҢеҸҜд»ҘзІҫзЎ®жҺ§еҲ¶д»»еҠЎзҡ„жү§иЎҢй—ҙйҡ”гҖҒеҒңжӯўжқЎд»¶е’ҢзәҝзЁӢгҖӮ

**д»Јз Ғпјҡ**



```
using Cysharp.Threading.Tasks;
using System;
using System.Threading;
using UnityEngine;

public static class LoopTaskUtils
{
    /// <summary>
    /// ејӮжӯҘжү§иЎҢдёҖдёӘе‘ЁжңҹжҖ§д»»еҠЎпјҢзӣҙеҲ°ж»Ўи¶іеҒңжӯўжқЎд»¶жҲ–иў«еҸ–ж¶ҲгҖӮ
    /// </summary>
    /// <param name="intervalSeconds">жҜҸж¬Ўиҝӯд»Јд№Ӣй—ҙзҡ„й—ҙйҡ”ж—¶й—ҙпјҲз§’пјүгҖӮ</param>
    /// <param name="action">жҜҸж¬Ўиҝӯд»Јжү§иЎҢзҡ„еҠЁдҪңгҖӮ</param>
    /// <param name="stopCondition">еҒңжӯўеҫӘзҺҜзҡ„жқЎд»¶еҮҪж•°гҖӮиҝ”еӣһ true еҲҷеҒңжӯўгҖӮ</param>
    /// <param name="cancellationToken">еҸҜйҖүзҡ„еҸ–ж¶Ҳд»ӨзүҢгҖӮ</param>
    /// <param name="runOnMainThread">д»»еҠЎжҳҜеҗҰеңЁдё»зәҝзЁӢжү§иЎҢгҖӮеҰӮжһңдёә falseпјҢaction дјҡеңЁеҗҺеҸ°зәҝзЁӢжү§иЎҢгҖӮ</param>
    /// <returns>UniTaskгҖӮ</returns>
    public static async UniTask DoPeriodicTask(
        float intervalSeconds,
        Action action,
        Func<bool> stopCondition = null,
        CancellationToken cancellationToken = default,
        bool runOnMainThread = true)
    {
        try
        {
            while (!cancellationToken.IsCancellationRequested && (stopCondition == null || !stopCondition()))
            {
                if (!runOnMainThread)
                {
                    await UniTask.SwitchToThreadPool();
                }
                
                action?.Invoke();

                if (!runOnMainThread)
                {
                    await UniTask.SwitchToMainThread();
                }

                if (!cancellationToken.IsCancellationRequested && (stopCondition == null || !stopCondition()))
                {
                    await UniTask.Delay(TimeSpan.FromSeconds(intervalSeconds), false, PlayerLoopTiming.Update, cancellationToken);
                }
            }
        }
        catch (OperationCanceledException)
        {
            Debug.Log("е‘ЁжңҹжҖ§д»»еҠЎиў«еҸ–ж¶ҲгҖӮ");
        }
        catch (Exception ex)
        {
            Debug.LogError($"е‘ЁжңҹжҖ§д»»еҠЎеҸ‘з”ҹй”ҷиҜҜ: {ex.Message}");
        }
        finally
        {
            Debug.Log("е‘ЁжңҹжҖ§д»»еҠЎз»“жқҹгҖӮ");
        }
    }
}

```

**и®ҫи®Ўж„Ҹеӣҫпјҡ**

-   **зҒөжҙ»зҡ„еҫӘзҺҜжҺ§еҲ¶пјҡ**
    
    -   `intervalSeconds` жҺ§еҲ¶жҜҸж¬Ўиҝӯд»Јзҡ„ж—¶й—ҙй—ҙйҡ”гҖӮ
        
    -   `action` е§”жүҳдҪҝеҫ—еҫӘзҺҜдҪ“йҖ»иҫ‘еҸҜе®ҡеҲ¶гҖӮ
        
    -   `stopCondition` е§”жүҳе…Ғи®ёеҹәдәҺд»»ж„ҸеҠЁжҖҒжқЎд»¶пјҲеҰӮи®Ўж•°еҷЁиҫҫеҲ°дёҠйҷҗгҖҒзү№е®ҡзҠ¶жҖҒеҸҳеҢ–пјүиҮӘеҠЁз»ҲжӯўеҫӘзҺҜпјҢжҜ”з®ҖеҚ•зҡ„еӣәе®ҡж¬Ўж•°еҫӘзҺҜжӣҙејәеӨ§гҖӮ
        
-   **еҸҜйқ зҡ„еҸ–ж¶ҲжңәеҲ¶пјҡ** `while` еҫӘзҺҜеҶ…йғЁе§Ӣз»ҲжЈҖжҹҘ `cancellationToken.IsCancellationRequested`пјҢе№¶е°Ҷе…¶дј йҖ’з»ҷ `UniTask.Delay`пјҢзЎ®дҝқд»»еҠЎеҸҜд»ҘеңЁеӨ–йғЁиў«йҡҸж—¶еҸ–ж¶ҲпјҢйҒҝе…Қж— йҷҗеҫӘзҺҜжҲ–иө„жәҗжөӘиҙ№гҖӮ
    
-   **жҷәиғҪзҡ„зәҝзЁӢйҖүжӢ©пјҡ** `runOnMainThread` еҸӮж•°е…Ғи®ёдҪ еҶіе®ҡ `action` дёӯзҡ„йҖ»иҫ‘жҳҜеңЁдё»зәҝзЁӢиҝҳжҳҜеҗҺеҸ°зәҝзЁӢжү§иЎҢгҖӮеҰӮжһңи®ҫзҪ®дёә `false`пјҢеҲҷдјҡеңЁжү§иЎҢ `action` еүҚеҗҺиҮӘеҠЁиҝӣиЎҢзәҝзЁӢеҲҮжҚўпјҢзЎ®дҝқ `action` зҡ„**иҖ—ж—¶и®Ўз®—йғЁеҲҶ**еҸҜд»ҘеңЁеҗҺеҸ°зәҝзЁӢиҝҗиЎҢпјҢе№¶дё”д№ӢеҗҺе®үе…Ёең°еӣһеҲ°дё»зәҝзЁӢгҖӮ
    
-   **еҶ…е»әзҡ„й”ҷиҜҜеӨ„зҗҶпјҡ** еҶ…зҪ® `try-catch` еқ—жҚ•иҺ· `OperationCanceledException` е’Ңе…¶д»– `Exception`пјҢдёәе‘ЁжңҹжҖ§д»»еҠЎжҸҗдҫӣз»ҹдёҖзҡ„й”ҷиҜҜеӨ„зҗҶпјҢжҸҗеҚҮзЁіе®ҡжҖ§гҖӮ
    
-   **`Forget()` еҸӢеҘҪпјҡ** иҝҷз§Қе‘ЁжңҹжҖ§д»»еҠЎйҖҡеёёжҳҜвҖңеҗҜеҠЁеҗҺеҚіиҝҗиЎҢвҖқпјҢи°ғз”Ёж–№дёҚе…іеҝғе…¶иҝ”еӣһеҖјпјҢеёёз»“еҗҲ `Forget()` дҪҝз”Ёд»ҘйҒҝе…ҚиӯҰе‘ҠгҖӮ
    

----------

### 4. дёҺ MonoBehaviour з”ҹе‘Ҫе‘Ёжңҹз»‘е®ҡзҡ„д»»еҠЎеҗҜеҠЁеҷЁпјҡ`TaskRunner`

иҝҷдёӘе·Ҙе…·зұ»жҸҗдҫӣдәҶдёҖз§Қдјҳйӣ…зҡ„ж–№ејҸпјҢе°Ҷ `UniTask` дёҺ `MonoBehaviour` зҡ„з”ҹе‘Ҫе‘Ёжңҹзҙ§еҜҶз»‘е®ҡпјҢд»ҺиҖҢе®һзҺ°д»»еҠЎзҡ„иҮӘеҠЁз®ЎзҗҶе’ҢеҸ–ж¶ҲгҖӮ

**д»Јз Ғпјҡ**



```
using Cysharp.Threading.Tasks;
using UnityEngine;
using System.Threading;
using System;

public static class TaskRunner
{
    /// <summary>
    /// еңЁжҢҮе®ҡ MonoBehaviour зҡ„з”ҹе‘Ҫе‘ЁжңҹеҶ…еҗҜеҠЁдёҖдёӘ UniTaskгҖӮ
    /// д»»еҠЎдјҡеңЁ MonoBehaviour й”ҖжҜҒж—¶иҮӘеҠЁеҸ–ж¶ҲгҖӮ
    /// </summary>
    /// <param name="monoBehaviour">з»‘е®ҡз”ҹе‘Ҫе‘Ёжңҹзҡ„ MonoBehaviour е®һдҫӢгҖӮ</param>
    /// <param name="taskFunc">иҰҒжү§иЎҢзҡ„ UniTask еҮҪж•°гҖӮ</param>
    public static UniTask RunOnDestroy(this MonoBehaviour monoBehaviour, Func<CancellationToken, UniTask> taskFunc)
    {
        return RunTaskWithLifecycle(monoBehaviour, taskFunc, monoBehaviour.GetCancellationTokenOnDestroy());
    }

    /// <summary>
    /// еңЁжҢҮе®ҡ MonoBehaviour зҡ„з”ҹе‘Ҫе‘ЁжңҹеҶ…еҗҜеҠЁдёҖдёӘ UniTaskгҖӮ
    /// д»»еҠЎдјҡеңЁ MonoBehaviour зҰҒз”Ёж—¶иҮӘеҠЁеҸ–ж¶ҲгҖӮ
    /// </summary>
    /// <param name="monoBehaviour">з»‘е®ҡз”ҹе‘Ҫе‘Ёжңҹзҡ„ MonoBehaviour е®һдҫӢгҖӮ</param>
    /// <param name="taskFunc">иҰҒжү§иЎҢзҡ„ UniTask еҮҪж•°гҖӮ</param>
    public static UniTask RunOnDisable(this MonoBehaviour monoBehaviour, Func<CancellationToken, UniTask> taskFunc)
    {
        return RunTaskWithLifecycle(monoBehaviour, taskFunc, monoBehaviour.GetCancellationTokenOnDisable());
    }

    private static async UniTask RunTaskWithLifecycle(MonoBehaviour monoBehaviour, Func<CancellationToken, UniTask> taskFunc, CancellationToken lifecycleToken)
    {
        if (monoBehaviour == null)
        {
            Debug.LogWarning("TaskRunner: MonoBehaviour е®һдҫӢдёәз©әпјҢд»»еҠЎж— жі•еҗҜеҠЁгҖӮ");
            return;
        }

        try
        {
            await taskFunc(lifecycleToken);
        }
        catch (OperationCanceledException)
        {
            if (monoBehaviour != null)
            {
                Debug.Log($"д»»еҠЎз»‘е®ҡеҲ° {monoBehaviour.name} з”ҹе‘Ҫе‘ЁжңҹпјҢеӣ еҸ–ж¶Ҳд»ӨзүҢи§ҰеҸ‘иҖҢз»“жқҹгҖӮ");
            }
        }
        catch (Exception ex)
        {
            if (monoBehaviour != null)
            {
                Debug.LogError($"д»»еҠЎз»‘е®ҡеҲ° {monoBehaviour.name} еҸ‘з”ҹжңӘеӨ„зҗҶзҡ„й”ҷиҜҜ: {ex.Message}");
            }
            else
            {
                Debug.LogError($"дёҖдёӘз»‘е®ҡз”ҹе‘Ҫе‘Ёжңҹзҡ„д»»еҠЎеңЁ MonoBehaviour й”ҖжҜҒеҗҺеҸ‘з”ҹжңӘеӨ„зҗҶзҡ„й”ҷиҜҜ: {ex.Message}");
            }
        }
    }
}

```

**и®ҫи®Ўж„Ҹеӣҫпјҡ**

-   **зӣҙи§Ӯзҡ„жү©еұ•ж–№жі•пјҡ** `RunOnDestroy` е’Ң `RunOnDisable` иў«и®ҫи®Ўдёә `MonoBehaviour` зҡ„жү©еұ•ж–№жі•пјҢеҸҜд»ҘзӣҙжҺҘеңЁд»»дҪ• `MonoBehaviour` е®һдҫӢдёҠи°ғз”ЁпјҢеҰӮ `this.RunOnDestroy(...)`пјҢжһҒеӨ§ең°жҸҗй«ҳдәҶд»Јз Ғзҡ„з®ҖжҙҒжҖ§е’Ңзӣҙи§ӮжҖ§гҖӮ
    
-   **иҮӘеҠЁз”ҹе‘Ҫе‘Ёжңҹз»‘е®ҡпјҡ** иҝҷжҳҜж ёеҝғеҠҹиғҪгҖӮе®ғеҲ©з”Ё `UniTask` жҸҗдҫӣзҡ„ `GetCancellationTokenOnDestroy()` е’Ң `GetCancellationTokenOnDisable()` жү©еұ•пјҢиҮӘеҠЁе°Ҷд»»еҠЎзҡ„з”ҹе‘Ҫе‘ЁжңҹдёҺ `MonoBehaviour` зҡ„й”ҖжҜҒжҲ–зҰҒз”ЁдәӢд»¶з»‘е®ҡпјҢйҳІжӯўеёёи§Ғзҡ„еҶ…еӯҳжі„жјҸе’Ңеғөе°ёд»»еҠЎй—®йўҳгҖӮ
    
-   **з»ҹдёҖд»»еҠЎеҗҜеҠЁдёҺй”ҷиҜҜеӨ„зҗҶпјҡ** `RunTaskWithLifecycle` дҪңдёәз§Ғжңүиҫ…еҠ©ж–№жі•пјҢиҙҹиҙЈе®һйҷ…зҡ„д»»еҠЎеҗҜеҠЁе’ҢйҖҡз”Ёзҡ„ејӮеёёжҚ•иҺ·гҖӮе®ғиғҪжҷәиғҪең°еҲӨж–ӯ `MonoBehaviour` е®һдҫӢжҳҜеҗҰд»Қз„¶еӯҳеңЁпјҢе№¶еңЁж—Ҙеҝ—дёӯжҸҗдҫӣжӣҙеҮҶзЎ®зҡ„дёҠдёӢж–ҮдҝЎжҒҜгҖӮ
    
-   **`Func<CancellationToken, UniTask>` еҸӮж•°пјҡ** иҝҷз§Қе§”жүҳзӯҫеҗҚејәеҲ¶дҪ дј е…Ҙзҡ„д»»еҠЎеҮҪж•°жҺҘж”¶дёҖдёӘ `CancellationToken` еҸӮж•°гҖӮиҝҷж ·пјҢдҪ зҡ„д»»еҠЎеҶ…йғЁе°ұеҸҜд»Ҙж–№дҫҝең°и®ҝй—®е№¶е“Қеә”еҸ–ж¶ҲиҜ·жұӮпјҢзЎ®дҝқеҚҸеҗҢеҸ–ж¶Ҳзҡ„жӯЈзЎ®е®һзҺ°гҖӮ
    
-   **йҖҡеёёдёҺ `Forget()` з»“еҗҲпјҡ** з»‘е®ҡз”ҹе‘Ҫе‘Ёжңҹзҡ„д»»еҠЎйҖҡеёёжҳҜвҖңеҗҜеҠЁеҗҺиҝҗиЎҢвҖқзҡ„пјҢи°ғз”Ёж–№дёҚе…іеҝғе…¶е®ҢжҲҗз»“жһңпјҢеӣ жӯӨеёёеёёдјҡй…ҚеҗҲ `Forget()` ж–№жі•дҪҝз”ЁпјҢд»ҘйҒҝе…Қзј–иҜ‘еҷЁзҡ„ `await` иӯҰе‘ҠпјҢеҗҢж—¶зЎ®дҝқд»»еҠЎеңЁеҗҺеҸ°е®үе…ЁиҝҗиЎҢгҖӮ
    

----------

## жҖ»з»“

йҖҡиҝҮдёҠиҝ°зҡ„дәҢж¬Ўе°ҒиЈ…зӨәдҫӢпјҢжҲ‘д»¬иғҪжё…жҘҡең°зңӢеҲ°пјҢеҜ№ `UniTask` иҝӣиЎҢе°ҒиЈ…дёҚд»…д»…жҳҜд»Јз ҒдёҠзҡ„з®ҖеҚ•еӨҚеҲ¶зІҳиҙҙпјҢжӣҙжҳҜдёҖз§Қ**и®ҫи®ЎжЁЎејҸ**зҡ„дҪ“зҺ°гҖӮе®ғи®©жҲ‘д»¬иғҪеӨҹпјҡ

1.  **жҸҗеҚҮд»Јз Ғзҡ„жҠҪиұЎеұӮзә§пјҡ** е°Ҷеә•еұӮејӮжӯҘз»ҶиҠӮжҠҪиұЎдёәжӣҙз¬ҰеҗҲдёҡеҠЎиҜӯд№үзҡ„й«ҳзә§жҺҘеҸЈгҖӮ
    
2.  **ејәеҲ¶жңҖдҪіе®һи·өпјҡ** еңЁе°ҒиЈ…еҶ…йғЁеөҢе…ҘзәҝзЁӢе®үе…ЁгҖҒејӮеёёеӨ„зҗҶгҖҒеҸ–ж¶ҲжңәеҲ¶зӯүжңҖдҪіе®һи·өпјҢйҒҝе…ҚејҖеҸ‘иҖ…йҒ—жјҸгҖӮ
    
3.  **жҸҗй«ҳеӣўйҳҹеҚҸдҪңж•ҲзҺҮпјҡ** з»ҹдёҖзҡ„жҺҘеҸЈе’Ңи§„иҢғи®©еӣўйҳҹжҲҗе‘ҳиғҪеӨҹжӣҙеҝ«дёҠжүӢпјҢе№¶дә§еҮәжӣҙй«ҳиҙЁйҮҸгҖҒжӣҙдёҖиҮҙзҡ„д»Јз ҒгҖӮ
    

жҺҢжҸЎ `UniTask` жң¬иә«жҳҜеҹәзЎҖпјҢиҖҢеӯҰдјҡеҰӮдҪ•ж №жҚ®йЎ№зӣ®йңҖжұӮеҜ№е…¶иҝӣиЎҢ**й«ҳеұӮж¬Ўзҡ„дәҢж¬Ўе°ҒиЈ…**пјҢеҲҷжҳҜдёҖдёӘз»ҸйӘҢдё°еҜҢзҡ„ Unity ејҖеҸ‘иҖ…иҝҲеҗ‘еҚ“и¶Ҡзҡ„е…ій”®дёҖжӯҘгҖӮ

еёҢжңӣиҝҷдәӣзӨәдҫӢе’Ңи§ЈйҮҠиғҪеё®еҠ©дҪ жӣҙеҘҪең°зҗҶи§Ј `UniTask` зҡ„дәҢж¬Ўе°ҒиЈ…еҸҠе…¶еёҰжқҘзҡ„е·ЁеӨ§д»·еҖјпјҒеңЁдҪ зҡ„йЎ№зӣ®дёӯпјҢжҳҜеҗҰиҝҳжңүе…¶д»–дҪ и®Өдёәзү№еҲ«йҖӮеҗҲиҝӣиЎҢ `UniTask` дәҢж¬Ўе°ҒиЈ…зҡ„еңәжҷҜе‘ўпјҹ
