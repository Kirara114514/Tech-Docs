### 1. 概念初探：点与向量的本源

在3D空间中，首先要区分两个最基本的概念：**点 (Point)** 和 **向量 (Vector)**。它们虽然都用 `(x, y, z)` 这样的坐标形式表示，但其几何意义和在游戏中的用途却截然不同。

#### 1.1 点：空间中的位置
**点**是3D空间中的一个具体**位置**，没有大小、没有方向，仅仅是空间中的一个占位符。在Unity中，通常使用 `Vector3` 类型来表示一个点。

#### 1.2 向量：方向与大小的结合
**向量**则是一个既有**方向 (Direction)** 又有**大小 (Magnitude)** 的量，描述的是一个**位移 (Displacement)** 或者一种**趋势**。向量没有固定的起点，只要方向和长度相同就是同一个向量。

### 2. 向量的基本运算

#### 2.1 向量的加法与减法
- **向量加法 (`A + B`)**: 两个位移的叠加，对应分量相加
- **向量减法 (`B - A`)**: 从点A到点B的位移，对应分量相减

#### 2.2 标量乘法与除法
- **标量乘法 (`k * A`)**: 改变向量的长度，k>1变长，0<k<1变短，k<0方向反转
- **标量除法 (`A / k`)**: 等同于乘以 `1/k`，将向量长度缩小k倍

#### 2.3 向量的模（长度）与单位向量
- **向量的模**: 向量的几何长度，`vector.magnitude` 返回实际长度
- **单位向量**: 长度为1的向量，只保留方向信息，`vector.normalized` 返回单位向量
- **性能优化**: 使用 `sqrMagnitude` 避免开方运算，在比较距离大小时性能更优

### 3. 向量的乘法：点积与叉积

#### 3.1 点积 (Dot Product)
- **计算方式**: 对应分量相乘后求和，结果是一个标量
- **几何意义**: 衡量两个向量方向的相似程度，`A·B = |A||B|cosθ`
- **应用场景**: 
  - 视野判断 (Field of View)
  - 光照计算
  - 判断物体是否面对彼此

#### 3.2 叉积 (Cross Product)
- **计算方式**: 结果是一个垂直于两个输入向量的新向量
- **几何意义**: 遵循右手定则，向量长度等于两个向量组成的平行四边形面积
- **应用场景**:
  - 计算表面法线
  - 判断左右关系
  - 构建旋转轴

### 4. 向量在Unity中的常见应用

#### 4.1 移动与位置控制
```csharp
// 沿着世界空间的X轴正方向每秒移动5个单位
transform.position += Vector3.right * 5f * Time.deltaTime;

// 沿着物体自身的"前方"方向每秒移动3个单位
transform.position += transform.forward * 3f * Time.deltaTime;
```

#### 4.2 计算距离与方向
```csharp
// 两点之间的距离
float distance = Vector3.Distance(player.position, enemy.position);

// 性能更优的平方距离
float sqrDistance = (player.position - enemy.position).sqrMagnitude;

// 朝向某个目标
Vector3 directionToTarget = target.position - transform.position;
Vector3 normalizedDirection = directionToTarget.normalized;
```

#### 4.3 物理交互与力学
```csharp
// 施加力
Rigidbody rb = GetComponent<Rigidbody>();
Vector3 pushDirection = transform.forward;
float forceMagnitude = 100f;
rb.AddForce(pushDirection * forceMagnitude);

// 反弹
Vector3 reflectedVector = Vector3.Reflect(incidentVector, surfaceNormal);
```

### 5. 常见问题与面试考点

1. **点与向量的区别**: 点表示绝对位置，向量表示相对位移或方向
2. **sqrMagnitude的性能优势**: 避免开方运算，在比较距离大小时性能更优
3. **点积的几何意义**: 衡量两个向量方向的相似程度，应用于视野检测等场景
4. **叉积的几何意义**: 得到垂直于两个输入向量的新向量，应用于计算法线、判断左右关系
5. **normalized操作的重要性**: 确保恒定速度移动，避免斜向移动速度变快

## 关键概念

- **点 (Point)**: 空间中的绝对位置
- **向量 (Vector)**: 既有方向又有大小的量，表示位移或趋势
- **向量的模 (Magnitude)**: 向量的长度
- **单位向量 (Normalized Vector)**: 长度为1的向量，只保留方向信息
- **点积 (Dot Product)**: 衡量两个向量方向的相似程度
- **叉积 (Cross Product)**: 得到垂直于两个输入向量的新向量
- **sqrMagnitude**: 向量长度的平方，性能优化技巧

## 技术要点

1. **性能优化**: 在只需要比较距离大小时使用 `sqrMagnitude` 而非 `magnitude`
2. **恒定速度移动**: 对输入向量进行 `normalized` 操作，确保移动速度一致
3. **视野判断**: 使用点积判断目标是否在视野范围内
4. **法线计算**: 使用叉积计算三角形表面的法线方向
5. **左右判断**: 结合点积和叉积判断目标在左侧还是右侧

## 总结

向量是3D空间中的"原子"，理解并熟练运用它是掌握更复杂3D数学概念的基础。本文系统讲解了向量的基本概念、运算规则和实际应用，为后续学习旋转、矩阵变换等高级概念打下坚实基础。掌握这些知识不仅能提高开发效率，还能在面试中自信回答相关问题。
