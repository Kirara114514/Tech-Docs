# C++智能指针使用心得

最近在重构代码时发现了一些内存管理的问题。

要点：
1. unique_ptr用于独占所有权
2. shared_ptr用于共享所有权
3. weak_ptr解决循环引用

需要注意：
- 避免裸指针和智能指针混用
- 注意shared_ptr的引用计数开销
- 使用make_shared而不是new
